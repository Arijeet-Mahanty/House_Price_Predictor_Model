# House Price Prediction - Linear Regression
This repository contains a machine learning project focused on predicting real estate prices. The project demonstrates a fundamental approach to regression analysis, from data preprocessing to model evaluation.
# 📌 Project Overview
The goal of this project is to build a predictive model that estimates house prices based on various features. By implementing a Linear Regression model, I analyze the relationship between independent variables (like square feet or location) and the target variable (price).
# 🛠️ Technologies & Tools
Python: The core programming language.

Pandas: Used for data manipulation and loading the dataset.

Scikit-learn:

LinearRegression: For building and training the model.

LabelEncoder: For converting categorical text data into numerical format.

# 📂 Project Workflow
Data Loading: Importing the dataset using Pandas.

Feature Encoding: Utilizing LabelEncoder to transform categorical variables into a format suitable for machine learning.

Data Splitting: Dividing the data into training and testing sets to evaluate performance.

Model Training: Training the Linear Regression model on the training data.

Prediction: Testing the model's accuracy on unseen data.

# 🚀 How to Use
Clone this repository: git clone https://github.com/Arijeet-Mahanty/house-price-prediction.git

Ensure you have the required libraries installed: pip install pandas scikit-learn matplotlib

Run the Python script or Jupyter/Colab notebook to see the model in action.
